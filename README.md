### Install
`npm i butter-installer-themes --save`

### Webpack
``` JS
 entry: [
   'butter-installer-theme',
   'index.js',
    ...
 ],
```

### CSS
``` CSS
 @import './node_modules/butter-installer-themes/index.css';
```

### HTML
``` HTML
 <link rel="stylesheet" type="text/css" href="./node_modules/butter-installer-themes/index.css">
```
